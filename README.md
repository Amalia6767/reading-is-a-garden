## 这是什么 · What it is

我是一个策展人，没有编程背景。读硬核论文对我来说很痛苦——化学、AI 论文常常读一两篇就不想读了。

问题从来不是"读不懂"，是**"读不下去"**。

市面上的工具都在帮你"读懂这一篇"（摘要、翻译、问答），但没有一个在解决"**明天还想不想再读一篇**"。就像 Duolingo 的厉害之处不是教得多好，而是让你连续 300 天回来——

**Reading is a Garden 想成为读论文界的 Duolingo。**

你读过的每一篇论文，都会在你的"知识花园"里长成一株植物。读得越多，花园越大。连续读，会有 streak 鼓励你别断签。目标不是让你读得多深，是让你**别弃坑**。

## 它怎么工作 · How it works

```
一篇论文 (PDF/字幕/文本)
   ↓
① 园丁零基础讲解 —— 为什么 → 是什么 → 最后才碰公式（配生活/类比）
   ↓
② 沉淀成花园里的一株植物 (markdown 笔记)
   ↓
③ 打卡：连续读了几天？🔥 别断签
```

## 快速开始 · Quick Start

```bash
# 1. 装依赖
pip install -r requirements.txt

# 2. 配置你的 API key
cp .env.example .env
# 然后编辑 .env，填入你的 DeepSeek key（https://platform.deepseek.com 申请）

# 3. 读一篇论文
python agent.py
# 把论文文件路径拖进去，园丁开始讲解

# 4. 逛你的花园
python garden_view.py
# 打开生成的「花园.md」，看看你种下了多少株植物
```

## 现在的状态 · Status

这是一个**正在公开生长**的项目 —— 我在一边学编程一边造它。

- ✅ **v0.1（现在）**：读论文 → 零基础讲解 → 存成花园植物 → 连续天数打卡

代码现在很简单，没什么难度。但难度和创意会**当着你的面一点点长出来**——这正是我想公开记录的东西：**一个零基础的人，如何从零造出一个 agent。**
